# Vector Visualizer

An interactive web application for visualizing and understanding linear algebra concepts

## Features

- **Interactive Vector Manipulation**: Click and drag vectors in 2D space
- **Linear Transformations**: Apply various matrix transformations with smooth animations
- **Basis Vectors Visualization**: See how î (i-hat) and ĵ (j-hat) transform
- **Preset Transformations**: Quick access to common transformations:
  - Rotations (45°, 90°)
  - Scaling (uniform and non-uniform)
  - Shearing (X and Y axes)
  - Reflections
  - Projections
- **Custom Matrix Input**: Define your own transformation matrices
- **Real-time Information**:
  - Current transformation matrix
  - Determinant calculation
  - Eigenvalues and eigenvectors
- **Multiple Vectors**: Add, remove, and manipulate multiple vectors

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Technology Stack

- React + TypeScript
- Vite
- TailwindCSS
- Canvas API for rendering
